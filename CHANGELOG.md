## v1.4.1 — 自动同步 + 离线支持

### ✨ 新增功能

- 📶 网络监听自动推送 — 离线期间喝的水，联网后自动推送到排行榜，无需手动操作
- 📲 PWA 安装恢复 — 重新启用 Service Worker，支持添加到手机桌面
- 📴 离线可用 — Service Worker 缓存页面文件，无网络时也能打开使用

### 🎨 优化

- 同步按钮永久变灰 — 已改为自动推送，不再需要手动点击
- Service Worker 更新机制 — 改 `SW_DEPLOY` 值即可触发缓存更新，不会再卡在旧版

### 🐛 修复

- 清理 `_sbClient` 残留引用 — 防止 `disconnectLeaderboard` 报错崩溃

## v1.4.0 — 日历交互 + 深色模式 + 排行榜周期

### ✨ 新增功能

- 📅 日历日期可点 — 点击热力图中任一天，弹窗查看当日每条喝水记录
- 📅 日历翻月 — 日历标题栏新增 ◀ ▶ 按钮切换月份
- 👅 喝一小口 (15ml) — 首页新增「喝一小口」按钮
- 🏆 排行榜本周/本月 — 数据库新增 `week_ml` / `month_ml` 字段，切换标签时正确显示累计排名
- 📋 关于页更新日志 — 关于应用弹窗内显示更新日志
- 🌙 深色模式完整样式 — 补全所有页面元素深色配色
- 🌙 auto 按钮 — 深色模式标签旁新增 `auto` 按钮
- ⏱ 提醒周期可调 — 可自定义 1-60 分钟，默认 30 分钟
- ⌚ 断开手表 — 设置页新增「断开手表」按钮

### 🐛 修复

- 深色模式无效果 — 补全完整的深色模式 CSS
- 刷新后通知不恢复 — 页面加载时主动请求授权
- 排行榜切换标签排名不变 — 改用对应的累计字段
- 首次加载无今日记录时崩溃 — `getTodayTotal` 空数组处理
- 深色模式开关不反映实际状态 — auto 模式下根据系统主题显示

### 🎨 优化

- 排行榜去掉 10 秒自动轮询 — 改为手动刷新
- 累计值缓存 — 增量更新，避免遍历全部历史记录
- 攒批推送 — 连续喝水攒 30 秒一批推送
- 撒花动画减半 — 粒子数 80→40，加冷却锁

### 🔧 技术改动

- 深色模式改为三态 — 跟随系统 / 开启 / 关闭
- 数据库迁移 — `leaderboard` 表新增 `week_ml` / `month_ml`
- CSS 变量化 — 品牌色定义在 `:root`，改主题色只改一处
- localStorage key 常量化 — 统一收到 `LS` 对象
- DOM 查询缓存 — 新增 `$()` 辅助函数
- 连续达标天数优化 — 先建日期映射表再计算
- XSS 防范 — 用户名改用 `data-` 属性 + 事件委托
- 应用图标更新 — 改用 SVG 矢量图标

## v1.3.0 — 零外部依赖 + 同步稳定

### 🐛 修复

- Safari "无法打开网页" — 移除外部的 CDN 依赖，不会被 Safari 内容拦截器阻拦
- Upsert 返回 409 不写入 — leaderboard 表主键从 id 改为 user_id

### 🔧 技术改动

- 移除 Supabase JS SDK — 全部改用原生 fetch
- 移除 Chart.js — 历史图表改用原生 Canvas 自绘
- 移除 Service Worker — 删除 sw.js 及其注册代码
- 清理遗留代码 — 删除 `_sbClient` 等 SDK 残留
