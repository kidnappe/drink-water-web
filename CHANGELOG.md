## v1.4.2 — 问题修复

### 🐛 修复

- 主页记录删除叉不靠右 — 布局修复
- 主页记录心情 + 按钮消失 — formatRecordLine 不包含交互按钮
- 月份柱状图日期挤在一起 — 改为按周间隔显示，加大底部留白
- 历史页柱状图日期消失 — fillText 坐标错误
- 管理员删除叉触发详情页 — 缺少 event.stopPropagation()
- 喝水日历有记录的日子看不见日期数字 — z-index 层级问题
- 关于页更新日志空白 — CHANGELOG.md 编码损坏
- 心情说明页显示原始 unicode 字符串 — JS 转义语法错误
- 喝一小口量不可调 — 新增 sipAmount 设置

### 🎨 优化

- PWA 图标：天蓝底 + 白色水杯
- SVG 矢量图标替换 PNG 图标
- CHANGELOG.md 改为 UTF-8 无 BOM 编码
- 新增 .gitattributes 强制 UTF-8
