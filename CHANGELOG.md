## v1.2.0 — 情绪记录 + 查看他人记录

### ✨ 新增功能

- **😊 情绪记录** — 喝水时可以选一个表情记录当前心情（😊😄🥰😌😤😢🤯💪）
- **👆 查看对方记录** — 排行榜上点击别人的头像，弹出框查看对方今日每条喝水记录（含情绪）

### 🔧 技术改动

- 每条喝水记录新增 `mood` 字段
- 同步到 Supabase 时附带 `today_records`（含每条记录的时间、水量、情绪）
- **需要运行 SQL：** `ALTER TABLE leaderboard ADD COLUMN today_records JSONB DEFAULT '[]'::jsonb;`

## v1.0.0 — 初始版本

### ✨ 功能

- 💧 **喝水记录** — 250/350/500/1000ml 一键记录 + 自定义水量
- 🪴 **养成花园** — 连续达标天数植物成长（种子→嫩芽→开花）
- 🏆 **排行榜** — 本地模拟排行榜（含 50 种动物头像 + 昵称）
- 📊 **统计图表** — 周/月柱状图（Chart.js）、喝水日历热力图
- ⚙️ **设置** — 每日目标、自定义水量、喝水提醒、深色模式、头像昵称
- 📤 **数据导入导出** — JSON 备份与恢复
- 🎉 **动画特效** — 达标撒花、水杯发光、波浪动画、溢出效果
